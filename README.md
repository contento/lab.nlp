## Setting the PIP environment

```shell
# [MacOS M*] 
# arch -x86_64 python3 -m venv .venv && \
# [Windows ] 
# ./.venv/scripts/activate && \
python -m venv .venv  && \
source .venv/bin/activate && \ 
python -m pip install -U -r requirements.txt && \
python -m pip install --upgrade pip 
```
